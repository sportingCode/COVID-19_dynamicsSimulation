- data folder includes the time-series data for the coronavirus disease (COVID-19) obtained from the COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) 
at Johns Hopkins University available at https://github.com/CSSEGISandData/COVID-19

- code folder includes the main.R script to launch and all the R functions needed collected in the "script" subfolder

- to change the inital configuration of the model, please go into code/script/input/config.R
